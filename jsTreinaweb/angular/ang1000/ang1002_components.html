Componentes

O angular trabalha com o cnceito de componentes, que são trechos de código que controlam a interface, incluindo controles como buttons, inputs, etc.

componentes precisam ser genéricos para que possam ser reutilizados.

Criando um componente manualmente

Na pasta app criar um novo componente: meu-componente.componente.ts

Depois registrar o componente no app.module.ts

Então incluir o componente no nosso HTML
