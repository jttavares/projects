O que são Frameworks?


Frameworks são aplicações que nos ajudam a criar outras aplicações.
Eles já vêm com várias funcionalidades prontas que resolvem as principais necessidades que surgem durante a criação de uma aplicação, como gerenciamento de rotas, criação de componentes, gerenciamento de dados, etc.
A principal diferença com uma biblioteca é que os nossos códigos usam as bibliotecas. Podemos chamar as funcionalidades das bibliotecas de qualquer lugar como quisermos.
Já os frameworks normalmente definem a estrutura e fluxo de uma aplicação, e é o framework que irá chamar as nossas funções.
Os Frameworks JavaScript mais conhecidos são: Angular, Vue.js, Ember, etc.
