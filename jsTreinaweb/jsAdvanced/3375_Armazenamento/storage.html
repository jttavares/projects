Podemos fazer vários tipos de aplicações com HTML e JavaScript. Mas quando atualizamos a página ou fechamos o navegador, as variáveis são reiniciadas.
Chega um momento em que as aplicações precisam se lembrar dos valores, mesmo que o usuário feche a aplicação.
Podemos então criar uma conexão com um servidor e armazenar os dados em um Banco de Dados. Para isso o usuário irá precisar de um cadastro e Login, além de só poder usar a aplicação quando estiver com conexão com a Internet.
Em certos casos pode ser que um servidor e um cadastro não sejam necessários para a sua aplicação, e que você também queira que o usuário possa usar a aplicação mesmo quando estiver sem Internet.
Para isso temos maneiras de armazenar dados no navegador.

Salvando Objetos

var myObj={
name:'Treinaweb',
curso: 'JavaScript - Avançado',
secao: 'LocalStorage'
}
localStorage.setItem('salvandoObjeto',JSON.stringify(myObj))
