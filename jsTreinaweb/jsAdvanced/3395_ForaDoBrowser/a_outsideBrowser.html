O JavaScript além do Navegador


O JavaScript inicialmente foi criado como um “extra” para o navegador. Seria apenas para podermos criar algumas funcionalidades simples, como validar um formulário antes de enviar os dados para o servidor.
Conforme foi evoluindo, o JavaScript se tornou uma linguagem poderosa e independente do navegador. Com isso pudemos começar a usar o JavaScript em vários lugares.
Vamos conhecer algumas áreas onde podemos atuar com JavaScript além do seu uso comum e até além dos navegadores. Dessa maneira você poderá usar seus conhecimentos em JavaScript e escolher entrar em alguma área além da web, como desenvolvimento de jogos, robótica, automação, etc.
Aqui no TreinaWeb há cursos relacionados a muitos dos assuntos que veremos!
