Introdução

Olá! Seja bem-vindo ao curso de Node.js da TreinaWeb! É um prazer ter você como nosso aluno.
Este curso tem como objetivo lhe mostrar o poder e facilidade que a plataforma Node.js traz para nós como desenvolvedores.
Neste curso, nós vamos desenvolver as principais funcionalidades que uma aplicação comum necessita disponibilizar: nós vamos manipular bancos de dados, vamos trabalhar com manipulação de arquivos e vamos também aprender a expor os recursos de nossa aplicação no ambiente Web.
Vamos também utilizar um dos mais tradicionais frameworks para desenvolvimento de aplicações Web utilizando a plataforma Node.js: o Express.
Por fim, vamos ainda aprender a fazer o deploy de nossa aplicação em um ambiente em nuvem – o Heroku, disponibilizando a nossa aplicação para o mundo!
Se você já fez outros cursos aqui da TreinaWeb, certamente já ouviu isso, mas nós gostaríamos de reforçar: qualquer dúvida ou sugestão que você tiver, por favor, entre em contato conosco através do canal de suporte.
Um ótimo curso para você! ;)
