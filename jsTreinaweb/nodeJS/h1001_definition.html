Afinal, o que é o Node.js?
É cada vez mais recorrente o Node.js surgir como opção de desenvolvimento de novas aplicações Web ao lado de opções mais tradicionais, como o Java e o ASP.NET. Mas, pelo fato de ser uma tecnologia relativamente nova frente estas outras tecnologias mais tradicionais citadas, muitas pessoas ainda não entendem muito bem o que de fato é o Node.js...
Antes de prosseguirmos, vamos esclarecer melhor: final de contas, o que é o Node.js!?
